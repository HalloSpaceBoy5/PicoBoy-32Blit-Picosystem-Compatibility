To Compile:
- Navigate to your build folder
- Run "cmake -DPICO_SDK_PATH=/path/to/pico/sdk -DPICOSYSTEM_DIR=/path/to/picosystem/sdk .."
- Modifiy DPICO_SDK_PATH and DPICOSYSTEM_DIR to reflect the locations of both the pico sdk and the picosystem sdk folders
- Run "make"

Disclaimer: Due to the hardware differences between the PicoBoy and other consoles, some games may not be compatable with the PicoBoy
